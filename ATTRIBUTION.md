# Attribution

This repository is a fork of [DioAugust/ws_dio_entrevistador](https://github.com/DioAugust/ws_dio_entrevistador), originally created by DioAugust as a workshop for building an OpenClaw skill for technical interviews.

## Adaptations for DEV.to OpenClaw Challenge (April 2026)
The following enhancements were added to better fit the challenge requirements and demonstrate advanced OpenClaw capabilities:

- **Bilingual Support**: Automatic language detection (Portuguese/English) based on job descriptions, with manual switching via commands like "switch to English".
- **Detailed Scoring**: Upgraded to a 1-5 scale with sub-scores for technical, behavioral, and domain competencies in JSON outputs.
- **Iterative Feedback Loops**: Added options during interview simulations to refine questions based on responses, improving interaction quality.
- **Enhanced JSON Structures**: Restructured outputs for easier API integration (e.g., added fields like `idioma_principal`, `dificuldade_estimada`, `feedback_sugestao`, and sub-scores).

These adaptations transform the original workshop into a more robust, challenge-ready tool while preserving the core educational value.
